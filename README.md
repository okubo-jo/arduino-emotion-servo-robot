# 感情によってサーボモーターの動作を変える

## 動画
![動画を見る](emotion-servo-robot.mp4)

## 概要
ボタン押すとOLEDディスプレイに表示される表情が切り替わり、その表情によってサーボモーターの動きを変化させるArduino作品を作りました。

- OLEDに顔（表情）を表示
- ボタンで感情を切り替え
- 感情ごとにサーボモーターの動きが変化

## 機能

### 喜び（Happy）
- 丸い目を表示
- サーボが左右にゆっくり動く

### 怒り（Angry）
- つり目で怒りを表現
- サーボが素早く左右に動く

### 眠い（Sleepy）
- 横線の目で眠いを表現
- サーボが停止する

## 使用機材
- Arduino
- OLEDディスプレイ（SSD1306 / I2C）
- サーボモーター
- タクトスイッチ
- ジャンパーワイヤー
- ブレッドボード

## 配線

| 部品 | 接続先 |
|------|--------|
| OLED VCC | 5V |
| OLED GND | GND |
| OLED SDA | A4 |
| OLED SCL | A5 |
| ボタン | D2 |
| サーボ信号 | D9 |

※ ボタンは `INPUT_PULLUP` を使用し、GNDに接続します。

## 工夫した点
- 表情とサーボの動きを連動させて感情を表現した
- シンプルな図形で軽量に顔を描画した
- ボタン1つで直感的に操作にした
- 感情ごとに動きのスピードを変化させた
